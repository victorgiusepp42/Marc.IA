# MarcIA
MarcIA— Tutor Interativo de Python com IA  Aplicação web desenvolvida em Flask que utiliza a API da Anthropic (Claude) para ensinar Python de forma conversacional, visando auxiliar calouros da UFCAT em matérias como Introdução a progrmação e áreas correlatas. O usuário digita dúvidas ou cola trechos de código, e o tutor responde com explicações, correções e exercícios progressivos adaptados ao nível do aluno.
